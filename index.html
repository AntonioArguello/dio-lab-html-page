Texto e Marcação

🔝 Voltar ao topo

A tag <mark> destaca um trecho importante do texto.

A tag <small> serve para texto menos importante.

A tag <i> deixa o texto em itálico, enquanto <u> deixa sublinhado.

Já <strong> dá ênfase forte: texto importante.

Podemos combinar tags, como todas juntas.

Listas Ordenadas e Não Ordenadas

🔝 Voltar ao topo

Lista Ordenada:

  1. <ol> cria uma lista ordenada
  2. <li> define os itens da lista

Lista Não Ordenada:

  • <ul> cria uma lista não ordenada
  • <li> é usada da mesma forma

Outras Tags Úteis

🔝 Voltar ao topo

A tag <a> cria links, como este: MDN Web Docs.

A tag <sub> mostra texto como em H2O (subscrito), e <sup> como em x2 (sobrescrito).

A tag

<blockquote>
é usada para citações:

"HTML é a espinha dorsal da web." — Desenvolvedor Anônimo

A tag <font> altera a aparência do texto (embora esteja obsoleta em HTML5).

A tag <del> indica texto removido ou modificado.

A tag <abbr> define abreviações como HTML.
